LEV / E-bike Conversion Stack

Wireless control, throttle mapping, cadence sensing, and power-aware behavior

HOOK is a bicycle conversion kit that turns a regular bicycle into an electric bicycle in seconds. The motor sits on the tire and uses friction drive, so the platform is fast to install, easy to remove, and practical for field use.

HOOK DAILY uses wired throttle control for a simple and reliable riding experience. HOOK UNIQ uses a wireless throttle via ESP-NOW, which expands the platform into more advanced use cases such as physiotherapy-style resistance control, gamified riding, and smart-trainer workflows with Zwift- or Blync-style compatibility.

HOOK DAILY → VESC integration with wired throttle control
HOOK UNIQ → ESP-NOW wireless throttle and drive-mode switching
Cadence sensing → nRF52832-A + BMA400 for pedal RPM and speed
System goal → robust, rider-friendly, field-ready operation
Design direction → mobility platform, resistance control, smart training, and real-world usability
